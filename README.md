# Communication Notebook
## Overview

保育園の連絡帳アプリの練習版です。

## Description

- 今回は認証機能は導入していないので、入力したidをContext APIを使ってグローバルなstateとして扱い、ログインをしているかのような動きを取り入れています
- Delete以外の基本的なCRUD処理の動きを導入しています。具体的には連絡帳の作成、更新、過去の連絡帳の取得などができます
